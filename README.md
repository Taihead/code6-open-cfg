# Открытый репозиторий конфигов code6.ru
Конфиги включают в себя список телепортов, локаций, аддон-машин и т.п. для того, чтобы сообщество могло оперативнее интегрировать изменения и вносить исправления вместе.

Правки принимаются через Pull Request с описанием изменений (потребуется аккаунт GitHub).
